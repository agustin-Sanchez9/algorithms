Given an integer "columnNumber", return its corresponding columns title as it appears in an excel sheet.

For example:
    A -> 1
    B -> 2
    ...
    Z -> 26
    AA -> 27
    ...
    ZY -> 701


Example 1:
    Input: columnNumber = 1
    Output: "A"

Example 2:
    Input: columnNumber = 28
    Output: "AB"

Example 3:
    Input: columnNumber = 701
    Output: "ZY"
